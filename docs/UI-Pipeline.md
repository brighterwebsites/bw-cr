# Pipeline UI

Nav: Pipeline

## Create project
- **+ New project** opens slide-in panel
- Customer: **Assign existing** or **Create MVP** (lead with business name + optional contact)
- Pick **pipeline stage** from dropdown (no forward/back buttons)
- Editable: name, scope, proposal, notes, start, deadline, completed (when Closed)

## Panel (edit)
- All project fields editable + Save
- **Tasks** mini-table: Status (inline) · Title · Due badge — **Open in Tasks** pre-fills search with project name
- **Assets** mini-table: Name · URL · Open URL · Open record (Assets page)

## Board
Stage/step columns from `project_stages`; click row opens panel. Slide-in width: `min(50%, 960px)`.
